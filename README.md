# zenbu.do

To-do management Apps with sleek and smooth UI.

Prerequisites:
1. Latest Node.js
2. Latest Browser (Recommended using chrome or Firefox)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
